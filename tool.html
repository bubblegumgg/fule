Custom Maya Python Tools for Asset Preparation

A set of custom Maya Python tools developed to streamline asset preparation and reduce repetitive manual work during environment production. These tools focus on pivot alignment, grid placement, and visual differentiation of assets to support efficient scene assembly and workflow testing.

Maya Python

01. Bottom Pivot Alignment & Grid Snapping Tool

A Maya Python tool that automatically aligns an object’s pivot to its lowest point and snaps both the pivot and the mesh to the grid. This ensures consistent base alignment and centered placement, making modular asset layout and engine integration more reliable.

02. Random Color Assignment Tool for Visual Differentiation

A Maya Python tool that assigns randomized colors to selected objects, allowing quick visual differentiation when working with multiple assets. This is useful for layout blocking, selection verification, and early-stage scene organization.
